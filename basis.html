#在 GitHub 建立一個「沒有版本」的空白 Git 儲存庫，然後透過 git clone 取得遠端儲存庫，再建立版本後上傳
        1.先去 githup 建立 new repository
        2.回到專案(終端機)鍵入   
            
            git clone https://github.com/sstudentr123/gitTutorial.git
            
            cd gittutorial
            
            echo a > a.text
            git add .
            git commit -m 'initial commit'
            git push -u  origin master
