        2.回到專案(終端機)鍵入

            git commit -m 'initial commit'
            
            git push -u  origin master


        #本地 Git 儲存庫上傳到指定的 GitHub 專案
            1.先在本地(終端機)鍵入 git init 
            2.後再新增檔案，提交添加到索引的檔案
            3.在指定remote add origin遠端儲存庫在哪 https://github.com/sstudentr123/test.git
                git init
                echo "# test" >> README.md
                git add README.md
                git commit -m "first commit"
                git remote add origin https://github.com/sstudentr123/test.git
                git push -u origin master
